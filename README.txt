----------------------------------------------
PROJECT DESCRIPTION
----------------------------------------------
This web application is a Random Quote Generator that retrieves quotes from
a public API (ZenQuotes.io) and displays them to the user. It was developed
as part of our IT110 Midterm Examination to demonstrate our ability to:

• Use HTML, CSS, and JavaScript to build a responsive, interactive website.
• Fetch data from a public API using asynchronous JavaScript (fetch + async/await).
• Handle potential API or network errors gracefully.
• Work collaboratively as a team to divide tasks and produce a polished output.

----------------------------------------------
API USED
----------------------------------------------
• Source: https://zenquotes.io/api/random
• Proxy: https://api.allorigins.win/get?url=
(The proxy is used to bypass CORS restrictions when fetching data.)

----------------------------------------------
HOW TO RUN THE PROJECT
----------------------------------------------
1. Download or unzip the project folder.
2. Open the file “index.html” in a modern web browser (e.g., Chrome, Edge, Firefox).
3. On the portfolio page, click the button “Open Random Quote Generator.”
4. The app page will open. Click “New Quote” to fetch a random quote.
5. If the internet connection fails, an error message will appear instead of a quote.

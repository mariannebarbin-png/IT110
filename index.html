Group 11 - IT110 Midterm Project

Project: Random Quote Generator (using external API)

We are Group 11 of BSIT students from Caraga State University, passionate about learning web development and API integration. This portfolio showcases our teamwork, creativity, and problem-solving skills as we built a dynamic Random Quote Generator.

Project Description

This project is a Random Quote Generator that fetches inspiring quotes from a public API and displays them to the user. It demonstrates our understanding of JavaScript's fetch and async/await features, error handling, and interactive web design.

  • Main API used: https://zenquotes.io/api/random
  • Files included: index.html, app.html, style.css, script.js
